# Image‑Resizing

**Image‑Resizing** はブラウザだけで動作する軽量な画像リサイズツールです。  
ファイルをアップロードしてもサーバーに送信されることはなく、すべてローカルのメモリ上で完結します。

> 👉 **デモ（GitHub Pages）**: <https://kan-taro.github.io/Image-Resizer/>  
> 👉 **ソースコード (GitHub)**: <https://github.com/kan-taro/Image-Resizer/>

---

## ✨ 特徴

| # | 内容 |
|---|------|
| 1 | **サーバー不要** – クライアント側だけで完結。プライバシー保護に最適です。 |
| 2 | **高品質リサイズ** – PICA (v8.0.0) を使用し、アンチエイリアス付きで高速に変換。 |
| 3 | **レスポンシブ UI** – Tailwind CSS によるシンプル＆モダンなデザイン。 |
| 4 | **GIF → PNG 自動変換** – GIF は静止画として PNG に変換してダウンロードします。 |
| 5 | **クロスブラウザ対応** – Chrome / Edge / Firefox / Safari 等でテスト済み（モバイルもOK）。 |
